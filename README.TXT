PROJECT TITLE:
Ursos

PURPOSE OF PROJECT:
Projeto de exemplo para o tutorial. Ursos se movem para frente
e comem folhas que encontram no caminho.

VERSION or DATE:
1.0

HOW TO START THIS PROJECT:
Compile todas as classes e clique no botão executar.

AUTHORS:
Michael Kolling
- Traduzido por Julio César Alves

USER INSTRUCTIONS:
Você precisará adicionar alguns ursos para que alguma coisa aconteça.
Clique na classe Ursos para instanciar novos ursos. Você pode também
adicionar folhas no mundo para os ursos comerem.
